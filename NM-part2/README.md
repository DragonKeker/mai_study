### Численные методы, 2 семестр

##### Пререквизиты

На машине, на которой будет запущен проект, должны быть установлены
* docker (>=19.03.5-beta1)
* docker-compose (>=1.23.0)
* node.js (>=13.10.1)
* npm (>=6.13.7)
* angular (>=9.0.6)

##### Backend
Запуск
```
cd back && ./run_app
```

##### Frontend
Запуск
```
cd front && ng serve
```

Компоненты лаб лучше объединить в один компонент.
